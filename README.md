# 📱 Employee Tracking App

A **React Native mobile application** designed to track employee activities, manage tasks, and monitor real-time performance for field workers and teams.

This app helps organizations replace manual tracking systems (Excel, calls, WhatsApp) with a **smart, automated solution**.

---

## 🚀 Features

### 👨‍🔧 Employee App
- 🔐 Authentication (Login / Signup)
- 📋 View assigned tasks
- 📍 Real-time location tracking (GPS)
- 🕒 Check-in / Check-out (Attendance)
- 🧾 Upload bills / reports (image)
- 📊 Daily work logs

---

### 👨‍💼 Admin Features
- 👥 Manage employees
- 📌 Assign tasks
- 📍 Track live location
- 📊 Monitor performance
- 📂 View uploaded reports

---

## 🛠️ Tech Stack

- **Frontend:** React Native (Expo)
- **Backend:** Supabase
- **Database:** PostgreSQL (via Supabase)
- **Storage:** Supabase Storage
- **Navigation:** React Navigation
